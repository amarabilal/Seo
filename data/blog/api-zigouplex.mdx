Zigouplex simplifie la synchronisation et la gestion des appareils IoT grâce à une architecture optimisée.

Exemple d’utilisation :

- **Villes intelligentes** : Gestion des feux de circulation en fonction du trafic.
- **Maisons connectées** : Coordination entre les appareils domestiques.

## Configuration initiale

### 1. **Obtenir votre clé API**

Pour commencer, vous devez générer une clé API depuis votre compte Zigouplex. Suivez ces étapes :

1. Connectez-vous à votre tableau de bord Zigouplex sur [zigouplex.com](https://zigouplex.com).
2. Accédez à la section **API Keys**.
3. Cliquez sur **Generate New Key** et copiez la clé.

### 2. **Configurer l’environnement**

Ajoutez votre clé API dans votre environnement de développement :

**Fichier `.env` :**

```env
ZIGOUPLEX_API_KEY=VOTRE_CLE_API
```

### 3. **Première requête**

Voici un exemple en Node.js pour vérifier la connexion à l’API :

```javascript
const axios = require('axios')

const API_URL = 'https://api.zigouplex.com/v1'
const API_KEY = process.env.ZIGOUPLEX_API_KEY

async function testConnection() {
  try {
    const response = await axios.get(`${API_URL}/status`, {
      headers: { Authorization: `Bearer ${API_KEY}` },
    })
    console.log('Statut de l’API :', response.data.status)
  } catch (error) {
    console.error('Erreur lors de la connexion à l’API :', error.message)
  }
}

testConnection()
```

## Cas d’utilisation détaillés

### **1. E-commerce**

Zigouplex aide les entreprises de commerce électronique à analyser les comportements d’achat et à personnaliser les recommandations.

#### Exemple :

Utilisez l’API pour suivre les interactions des utilisateurs avec votre site :

```javascript
async function trackUserActivity(userId, activity) {
  const response = await axios.post(
    `${API_URL}/analytics/track`,
    { userId, activity },
    {
      headers: { Authorization: `Bearer ${API_KEY}` },
    }
  )
  console.log('Activité enregistrée :', response.data)
}
trackUserActivity('user123', 'Ajouté un article au panier')
```

### **2. Sécurité**

Grâce à son module de reconnaissance faciale, Zigouplex est idéal pour les systèmes de sécurité.

#### Exemple :

Détecter un visage dans une image téléchargée :

```javascript
async function analyzeImage(imagePath) {
  const image = fs.readFileSync(imagePath, { encoding: 'base64' })
  const response = await axios.post(
    `${API_URL}/image/recognize`,
    { image },
    {
      headers: { Authorization: `Bearer ${API_KEY}` },
    }
  )
  console.log('Résultats de l’analyse :', response.data)
}
analyzeImage('./images/user_photo.jpg')
```

## Gestion des erreurs

Zigouplex renvoie des codes HTTP standard pour indiquer l’état des requêtes. Voici un aperçu des principaux codes :

- **200 OK** : La requête a réussi.
- **400 Bad Request** : Paramètres invalides.
- **401 Unauthorized** : Clé API manquante ou invalide.
- **500 Internal Server Error** : Problème côté serveur.

### Exemple de gestion d’erreurs :

```javascript
try {
  const response = await axios.get(`${API_URL}/endpoint`)
} catch (error) {
  if (error.response) {
    console.error('Erreur API :', error.response.status, error.response.data)
  } else {
    console.error('Erreur réseau :', error.message)
  }
}
```
